# Quick Start Guide - The Onslaught

Get up and running with "The Onslaught" mod in 5 minutes!

## Installation (1 minute)

1. **Download the mod** (already done if you're reading this)
2. **Copy** the `TheInfection` folder to your Project Zomboid mods directory:
   - **Windows**: `C:\Users\[YourName]\Zomboid\mods\`
   - **Linux**: `~/.local/share/Zomboid/mods/`
   - **Mac**: `~/Library/Application Support/Zomboid/mods/`
3. **Launch** Project Zomboid
4. **Enable** the mod in Mods menu
5. **Start** a new game (recommended)

## The Onslaught (18 Hours of Hell)

### Wave 1 (Hour 0)
**100 zombies spawn** around your base

**Actions**:
- Stay inside walls
- Monitor defenses
- Repair any damage
- Save energy

### Wave 2-4 (Hours 2, 4, 6)
**Increasing numbers** each wave

**Actions**:
- Rotate turret ammunition
- Replace broken traps
- Maintain barricades
- Stay alert

### Wave 5-9 (Hours 8-18)
**Peak difficulty** - largest waves

**Actions**:
- Focus on survival
- Conserve resources
- Don't take risks
- Watch for breaches

## Troubleshooting

**Q: Zombies spawning during setup phase?**
A: Restart game with mod properly enabled, check for conflicts

**Q: Onslaught not starting?**
A: Wait full duration, check for time acceleration mods

**Q: Can't craft defensive items?**
A: Need to learn recipes from skill books or trial and error

**Q: UI timer not showing?**
A: Check config or restart game

**Q: Too easy/hard?**
A: Adjust sandbox options before starting new game
